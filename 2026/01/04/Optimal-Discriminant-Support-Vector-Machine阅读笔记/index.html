《Optimal Discriminant Support Vector Machine》阅读笔记

发表于 2026-01-04 分类于 文献阅读
本文字数： 5.4k 阅读时长 ≈ 20 分钟

本文是对《Optimal Discriminant Support Vector Machine》这篇文章的阅读笔记，推导了二分类情景下文章所提模型目标函数的设计和求解过程。

Zhang, J., Lai, Z., Kong, H., & Yang, J. (2025). Learning the optimal discriminant SVM with feature extraction. IEEE Transactions on Pattern Analysis and Machine Intelligence, 47(4), 2897–2911. https://doi.org/10.1109/TPAMI.2025.3529711.

模型总览

ODSVM 将子空间学习和 SVM 分类放在一个统一的数学框架内共同进化。

投影矩阵：数据的特征提取器

\(P\) 是一个大小为 \(d \times m\) 的矩阵（\(d\) 是原始高维特征维度，\(m\) 是降维后的维度）。它的作用是将原始样本 \(x_i\) 投影到低维子空间： \[ z_i = P^\top x_i \] \(P\) 的目标在于丢弃大部分冗余信息的同时，精准地保留最有利于分类的信息。在 ODSVM 中，\(P\) 的构建受到 SVM 边距最大化的约束。这意味着 \(P\) 提取出的特征 \(z_i\) 必须让 SVM 能够更容易地找到一个宽阔的分类间隔（Margin）。

重构矩阵：数据的结构保持器

\(Q\) 同样是一个 \(d \times m\) 的矩阵，用于衡量投影后的低维特征 \(z_i\) 是否还保留了原始数据 \(x_i\) 的核心结构。数学上表现为最小化重构误差：\(\|x_i - Qz_i\|^2\)。 如果只追求分类准确率（只优化 \(P\) 和 SVM 参数），模型可能会陷入过拟合，或者提取出极其扭曲的特征。\(Q\) 的存在就像是一条绳子，拉住 \(P\)，要求提取的特征不仅要能分得开类别，还得能代表原本的数据。

迭代优化

在 ODSVM 的目标函数中，SVM 的参数（\(w, b\)）、投影矩阵 \(P\) 和重构矩阵 \(Q\) 是紧密耦合在一起的。由于无法直接求出所有变量的最优解，因此论文采用交替迭代优化（Alternating Optimization）的策略。

  1. 第一步：固定 \(P\) 和 \(Q\)，优化 SVM，找最强判别平面

    此时，\(P\) 已经把数据变成了低维特征 \(z_i\)。模型退化成一个标准的 SVM 问题。当前的子空间里，找到能把不同类别分得最开的超平面 \((w, b)\)。

  2. 第二步：固定 SVM 参数和 \(Q\)，优化 \(P\)

    \(P\) 需要同时满足两个要求：SVM 要求投影 \(z_i\) 得让它的分类间隔更大，\(Q\) 要求投影 \(z_i\) 要能配合重构矩阵更好地还原回 \(x_i\)。在这种约束下，\(P\) 会被微调，使得投影后的数据点在保持结构的同时，向着更有利于分类的方向偏移。

  3. 第三步：固定 \(P\) 和 SVM，优化 \(Q\)

    由于 \(P\) 已经变了，原来的恢复方案 \(Q\) 可能不再准确。这一步更新 \(Q\)，使得它能根据新的低维特征 \(z_i\) 更好地重建原始数据。

图正则化重构

ODSVM 并不是直接最小化低维点的距离，而是通过最小化邻居编码误差（Neighbor-encoding）来间接抑制低维空间中邻居间的距离。

构建有监督图

在进入迭代优化前，模型先为所有训练样本构建一个图 \(G\)，利用了类别信息和欧氏距离：

  • 同类近邻： 如果样本 \(x_i\) 和 \(x_j\) 属于同一类，且 \(x_j\) 是 \(x_i\) 的 \(k\) 个最近邻之一，那么就在它们之间连一条边，并赋予一个权重 \(S_{ij} > 0\)。
  • 异类或远邻： 如果类别不同或者距离太远，权重 \(S_{ij} = 0\)。

邻域编码

在构建子空间（学习 \(P\) 和 \(Q\)）时，目标函数不再仅仅是最小化 \(\|x_i - Q P^\top x_i\|^2\)，而是加入了基于图的约束。如果 \(S_{ij}\) 很大，模型会强制要求：

  • 在低维空间 \(z\) 中靠得更近： \(z_i\) 和 \(z_j\) 的距离必须尽可能小。

  • 共享重构特征： 它们的重构过程应该具有一致性。

对于投影矩阵 \(P\) 要求局部紧致性（Local Compactness）。\(P\) 在寻找投影方向时，会被要求最小化同类邻居在子空间中的方差。如果 \(P\) 选的方向让原本在一起的同类样本分开了，那么图正则化项就会产生很高的“惩罚值”。因此，\(P\) 提取出的特征能够保持数据的局部流形结构，即在原始高维空间里邻近的同类点，在低维子空间依然是邻居。

对于重构矩阵 \(Q\) 要求结构一致性。\(Q\) 不仅仅是一个数学上的逆变换，它被赋予了“平滑”数据的任务。通过考虑邻域， \(Q\) 学习到的是该类别数据在子空间中的分布规律，而不仅仅是单个点的坐标映射。因此，如果某个样本 \(x_i\) 含有噪声，但它的邻居们是干净的，邻域编码会利用邻居的结构信息来辅助 \(Q\) 的重构，从而“纠正”噪声样本在子空间中的偏离。

优化过程

以二分类为例，定义目标函数： \[ \min_{w, P, Q} \mathcal{J} = \underbrace{\frac{1}{2}\|w\|_2^2 + c \sum_{i=1}^n [1 - y_i w^\top P^\top x_i]_+}_{\text{SVM 分类项}} + \underbrace{\frac{\lambda}{2} \sum_{i,j=1}^n \|x_i - Q P^\top x_j\|_2^2 G_{ij}}_{\text{图正则化重构项}} + \underbrace{\frac{\gamma}{2}\|P\|_F^2}_{\text{正则化项}} \] 约束条件为：\(Q^\top Q = I\)。

其中，\(P\) 是投影矩阵，\(Q\) 是重构字典，\(w\) 是低维空间中的分类权重，\(G_{ij}\) 是刻画局部结构的图权重矩阵。\(c\) 用于平衡“最大化分类间隔”和“最小化分类错误”之间的权重，\(c\) 越大，对分类错误的容忍度越低。\(\lambda\) 用于控制“特征提取/结构保持”这一项在总目标函数中的重要性，\(\lambda\) 越大，模型越强调整体数据的结构保持和类内紧凑性。\(\gamma\) 用于控制投影矩阵的复杂度。

\([1 - y_i w^\top P^\top x_i]_+ = \max\left(0, \ 1 - y_i w^\top P^\top x_i\right)\) 是取正值的意思，在 SVM 中用来定义 Hinge Loss，若分对了且离边界够远，就不产生 Loss。分类错误或分类正确但离边界太近，损失就为正数，离正确边界越远，惩罚越大，迫使模型去修正参数。

  • \(\frac{1}{2}\|w\|_2^2 + c \sum [1 - y_i w^\top P^\top x_i]_+\) 是标准的 SVM 损失函数，但输入数据变成了 \(P^\top x_i\)，这意味着模型在寻找 \(P\) 和 \(w\) 使得数据在投影后能被最大间隔分开（最大化类间间隔）。
  • \(\frac{\lambda}{2} \sum \|x_i - Q P^\top x_j\|_2^2 G_{ij}\) 这一项利用邻居 \(x_j\) 的投影 \(P^\top x_j\)，通过 \(Q\) 映射回来去重构 \(x_i\)，同类样本在低维空间应该靠得很近，并且能保留原始数据的信息。这起到了“最小化类内散度”的作用。
  • \(\frac{\gamma}{2}\|P\|_F^2\) 是正则化项，防止投影矩阵 \(P\) 的值过大。

第一阶段：固定投影矩阵和重构字典，优化 SVM

当 \(P\) 和 \(Q\) 固定时，目标函数中与 \(P, Q\) 相关的重构项和正则项都变成了常数。此时问题退化为： \[ \min_{w} \frac{1}{2}\|w\|_2^2 + c \sum_{i=1}^n [1 - y_i w^\top (P^\top x_i)]_+ \] 这就是一个标准的线性 SVM 训练问题。目标是在投影后的低维子空间内，找到一个超平面系数 \(w\)，使得分类间隔（Margin）最大化。

第二阶段：固定 SVM 参数和重构字典，优化投影矩阵

因为 \(P\) 同时存在于分类项和重构项中。为了求解，作者引入松弛变量 \(\xi\) 并利用对偶理论。

首先进行目标函数重构，利用矩阵迹（Trace）的性质，重构项可以转化为： \[ \sum_{i,j=1}^n \|x_i - Q P^\top x_j\|_2^2 G_{ij} = Tr(P^\top S_A P) - 2 Tr(P^\top S_B Q) + \text{const} \] 其中 \(S_A = X D X^\top\) 和 \(S_B = X G X^\top\)，这两个矩阵被称为散射矩阵（Scatter Matrices）。\(S_A\) 通常与数据的全局能量或总方差有关，而 \(S_B\) 捕获了基于图结构的局部相关性。\(D\) 是度矩阵（Degree Matrix），是一个对角矩阵，其对角线上的元素 \(D_{ii}\) 是 \(G\) 矩阵第 \(i\) 行（或列）元素的总和： \[ D_{ii} = \sum_{j=1}^n G_{ij} \] 在矩阵运算中，迹 \(Tr(A)\) 是矩阵对角线元素之和。

向量或矩阵的二范数平方可以写成迹的形式：\(\|a\|_2^2 = a^\top a = Tr(a a^\top)\)。对于矩阵 \(A\)：\(\|A\|_F^2 = Tr(A^\top A) = Tr(AA^\top)\)。

矩阵的迹还有循环移位的性质：\(Tr(ABC) = Tr(BCA) = Tr(CAB)\)。如果一个运算的结果是标量，那么它的迹等于它本身：\(a^\top b = Tr(a^\top b)\)。

推导过程

首先利用向量范数公式 \(\|a - b\|_2^2 = a^\top a - 2 a^\top b + b^\top b\)，将目标函数展开为三项： \[ L = \sum_{i,j=1}^n \left( \underbrace{x_i^\top x_i}_{\text{第一项}} - \underbrace{2 x_i^\top (Q P^\top x_j)}_{\text{第二项}} + \underbrace{(Q P^\top x_j)^\top (Q P^\top x_j)}_{\text{第三项}} \right) G_{ij} \] 第一项是常数项，由于 \(x_i\) 和 \(D\) 在优化阶段都是已知的，这一项不包含变量 \(P\) 和 \(Q\)，因此记为 const： \[ \sum_{i,j=1}^n x_i^\top x_i G_{ij} = \sum_{i=1}^n \|x_i\|_2^2 \left( \sum_{j=1}^n G_{ij} \right) = \sum_{i=1}^n \|x_i\|_2^2 D_{ii} \] 在第二项中，由于 \(x_i^\top Q P^\top x_j\) 是一个标量，它等于自身的迹： \[ -2 \sum_{i,j=1}^n G_{ij} Tr(x_i^\top Q P^\top x_j) \] 利用迹的循环性质 \(Tr(ABC) = Tr(CAB)\)，将 \(P^\top x_j\) 移到前面： \[ -2 \sum_{i,j=1}^n Tr(P^\top x_j G_{ij} x_i^\top Q) \] 由于迹运算对求和是线性的，提取 \(P^\top\) 和 \(Q\)： \[ -2 Tr \left( P^\top \left( \sum_{i,j=1}^n x_j G_{ij} x_i^\top \right) Q \right) \] 由于 \(\sum_{i,j=1}^n x_j G_{ij} x_i^\top\) 正好是矩阵乘法 \(X G X^\top\) 的展开形式，定义 \(S_B = X G X^\top\)，则该项为： \[ -2 Tr(P^\top S_B Q) \] 第三项首先利用正交约束 \(Q^\top Q = I\) 简化内部项： \[ (Q P^\top x_j)^\top (Q P^\top x_j) = x_j^\top P \underbrace{Q^\top Q}_{I} P^\top x_j = x_j^\top P P^\top x_j \] \(Q\) 的作用是将低维特征 \(z\) “还原”回原始空间。如果没有 \(Q^\top Q = I\) 这个约束，模型可以通过无限缩小 \(P\) 并无限放大 \(Q\) 来保持乘积不变。这会导致计算极其不稳定。正交基意味着模型是在一个标准直角坐标系下进行重构，这保证了重构过程只是在寻找原始数据在子空间上的投影点，而不会对数据产生非线性的拉伸或剪切。

对于投影矩阵而言，\(P\) 的角色是寻找最有利于分类的特征，虽然 \(P\) 没有正交约束，但它有 \(\|P\|_F^2\) 正则化项。这里的正则化是 Frobenius 范数，把矩阵里所有的元素先平方、再求和、最后开根号。Frobenius 范数限制了投影矩阵 \(P\) 中元素的大小。如果没有这一项，\(P\) 的数值可能会为了迎合某些噪声数据而变得极大，导致模型泛化能力变差： \[ \|P\|_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^n p_{ij}^2} \] 回到公式的推导，代回原式并考虑对 \(i\) 的求和： \[ \sum_{i,j=1}^n (x_j^\top P P^\top x_j) G_{ij} = \sum_{j=1}^n (x_j^\top P P^\top x_j) \left( \sum_{i=1}^n G_{ij} \right) \] 利用度矩阵定义 \(\sum_i G_{ij} = D_{jj}\)： \[ \sum_{j=1}^n D_{jj} (x_j^\top P P^\top x_j) \] 转化为迹的形式并利用循环性质： \[ \sum_{j=1}^n D_{jj} Tr(x_j^\top P P^\top x_j) = Tr \left( P^\top \left( \sum_{j=1}^n D_{jj} x_j x_j^\top \right) P \right) \] 由于 \(\sum_j D_{jj} x_j x_j^\top\) 正好是矩阵乘法 \(X D X^\top\) 的对角加权形式，定义 \(S_A = X D X^\top\)，则该项为： \[ Tr(P^\top S_A P) \] 将三项合并，得到了论文中的简洁形式： \[ \sum_{i,j=1}^n \|x_i - Q P^\top x_j\|_2^2 G_{ij} = Tr(P^\top S_A P) - 2 Tr(P^\top S_B Q) + \text{const} \] 在固定 \(w\) 和 \(Q\) 后，ODSVM 的目标函数 \(\mathcal{J}\) 与变量 \(P\) 和松弛变量 \(\xi\) 相关的部分可以写成： \[ \min_{P, \xi} \mathcal{J}(P, \xi) = \underbrace{\frac{\lambda}{2} Tr(P^\top S_A P - 2 P^\top S_B Q)}_{\text{重构项}} + \underbrace{\frac{\gamma}{2}\|P\|_F^2}_{\text{P 的正则项}} + \underbrace{c \sum_{i=1}^n \xi_i}_{\text{SVM 误差项}} \] 约束条件来自于 SVM 的基本要求，即样本必须落在间隔之外或通过松弛变量补偿：\(y_i (w^\top P^\top x_i) \geq 1 - \xi_i, \quad \forall i=1, \dots, n\)，松弛变量非负：\(\xi_i \geq 0, \quad \forall i=1, \dots, n\)。

拉格朗日乘数法要求约束条件写成 \(g(x) \leq 0\) 的形式：\(1 - \xi_i - y_i w^\top P^\top x_i \leq 0\)，\(-\xi_i \leq 0\)。

拉格朗日函数的基本构造公式为：\(L = \text{目标函数} + \sum (\text{乘子} \times \text{约束})\)。

引入两组拉格朗日乘子：\(\alpha_i \geq 0\) 对应分类约束，\(\beta_i \geq 0\) 对应非负约束： \[ \begin{aligned} L(P, \xi, \alpha, \beta) &= \frac{\lambda}{2} Tr(P^\top S_A P - 2 P^\top S_B Q) + \frac{\gamma}{2}\|P\|_F^2 + c \sum_{i=1}^n \xi_i \\ &\quad + \sum_{i=1}^n \alpha_i (1 - \xi_i - y_i w^\top P^\top x_i) \\ &\quad + \sum_{i=1}^n \beta_i (-\xi_i) \end{aligned} \] 进一步把上式进行化简： \[ L(P, \xi, \alpha, \beta) = \frac{\lambda}{2} Tr(P^\top S_A P - 2 P^\top S_B Q) + \frac{\gamma}{2}\|P\|_F^2 + c e^\top \xi + \sum \alpha_i (1 - \xi_i - y_i w^\top P^\top x_i) \] \(c e^\top \xi\) 是对分类错误的惩罚总和，其中 \(e = [1, 1, \dots, 1]^\top\)，所以 \(e^\top \xi = \sum \xi_i\)。这里省略了 \(\beta_i\)项，是因为后续的求偏导过程有： \[ \frac{\partial L}{\partial \xi_i} = c - \alpha_i - \beta_i = 0 \implies \alpha_i + \beta_i = c \] 由于 \(\beta_i \geq 0\)，这个条件隐含了 \(0 \leq \alpha_i \leq c\)。因此，针对 \(P\) 优化拉格朗日函数时通常可以省略 \(\beta_i \xi_i\) 项，通过在后续求解 \(\alpha\) 的对偶问题时，给 \(\alpha_i\) 加上上限 \(c\)（即 \(0 \leq \alpha_i \leq c\)）来等价地处理 \(\xi_i \geq 0\) 的约束。

接着求导得出 \(P\) 的显式表达，令 \(\nabla_P L = 0\)，可以解出 \(P\) 关于 \(\alpha\) 的函数。

首先，从完整的拉格朗日函数中，剔除掉所有不包含 \(P\) 的项，因为求导后会变成0，得到关于 \(P\) 的子函数 \(L_P\)： \[ L_P = \underbrace{\frac{\lambda}{2} Tr(P^\top S_A P)}_{\text{项 1}} - \underbrace{\lambda Tr(P^\top S_B Q)}_{\text{项 2}} + \underbrace{\frac{\gamma}{2} Tr(P^\top P)}_{\text{项 3}} - \underbrace{\sum_{i=1}^n \alpha_i y_i w^\top P^\top x_i}_{\text{项 4}} \] 在推导过程需要用到以下三个矩阵求导公式：

  1. 公式 A： \(\frac{\partial Tr(P^\top A P)}{\partial P} = (A + A^\top) P\)。由于 \(S_A = XDX^\top\) 是对称矩阵，结果为 \(2 S_A P\)。
  2. 公式 B： \(\frac{\partial Tr(P^\top B)}{\partial P} = B\)。
  3. 公式 C： \(\frac{\partial (a^\top P^\top b)}{\partial P} = b a^\top\)。

将 \(\nabla_P L_P\) 分成四部分进行计算：

求导项 1：\(\frac{\partial}{\partial P} \left( \frac{\lambda}{2} Tr(P^\top S_A P) \right)\)

根据公式 A，由于 \(S_A\) 是对称的： \[ \nabla_P (\text{项 1}) = \frac{\lambda}{2} \cdot 2 S_A P = \lambda S_A P \] 求导项 2：\(\frac{\partial}{\partial P} \left( -\lambda Tr(P^\top S_B Q) \right)\)

根据公式 B，其中 \(B = S_B Q\)： \[ \nabla_P (\text{项 2}) = -\lambda S_B Q \] 求导项 3：\(\frac{\partial}{\partial P} \left( \frac{\gamma}{2} Tr(P^\top P) \right)\)

这等价于公式 A 中 \(A=I\) 的情况： \[ \nabla_P (\text{项 3}) = \frac{\gamma}{2} \cdot 2 I P = \gamma P \] 求导项 4：\(\frac{\partial}{\partial P} \left( -\sum_{i=1}^n \alpha_i y_i w^\top P^\top x_i \right)\)

这一步需要先进行矩阵化处理。

  1. 观察单个项：\(\alpha_i y_i w^\top P^\top x_i\)。这里 \(w\) 是分类权重，\(x_i\) 是样本。

  2. 利用公式 C，其中 \(a^\top = \alpha_i y_i w^\top\)， \(b = x_i\)。其导数为 \(x_i (\alpha_i y_i w^\top) = \alpha_i y_i x_i w^\top\)。

  3. 对所有 \(i\) 求和： \[ \nabla_P (\text{项 4}) = - \sum_{i=1}^n \alpha_i y_i x_i w^\top = - \left( \sum_{i=1}^n \alpha_i y_i x_i \right) w^\top \] 引入矩阵 \(Z = [y_1 x_1, y_2 x_2, \dots, y_n x_n] \in \mathbb{R}^{d \times n}\) 和向量 \(\alpha = [\alpha_1, \dots, \alpha_n]^\top\)，上面的求和项 \(\sum \alpha_i y_i x_i\) 正好等于矩阵乘积 \(Z \alpha\)。所以有： \[ \nabla_P (\text{项 4}) = - Z \alpha w^\top \]

将上述所有结果合并，得到总梯度： \[ \nabla_P L = \lambda S_A P - \lambda S_B Q + \gamma P - Z \alpha w^\top = 0 \] 化简后得到： \[ P = (\lambda S_A + \gamma I)^{-1} (\lambda S_B Q + Z \alpha w^\top) \] 这里包含三部分：

  • \(\lambda S_B Q\) 要求投影矩阵配合重构字典 \(Q\) 和图结构 \(S_B\)，尽量保留数据的原始几何信息
  • \(Z \alpha w^\top\) 要求投影矩阵参考 SVM 的分类结果，把不同类别的样本拉开，其通过拉格朗日乘子 \(\alpha\) 和权重 \(w\) 传递
  • \((\lambda S_A + \gamma I)^{-1}\) 则起到了一个“归一化”和“稳定器”的作用，确保 \(P\) 不会因为单方面的力量而数值爆炸

当得到了上述投影矩阵的显示表达后，接下来要求解针对 \(\alpha\) 的二次规划问题（QPP），需要将第二阶段求得的 \(P\) 的显式表达代回拉格朗日函数中，这个过程被称为对偶化（Dualization）。

首先对拉格朗日函数进行简化： \[ \begin{align*} L(P, \xi, \alpha, \beta) &= \frac{\lambda}{2} Tr(P^\top S_A P - 2 P^\top S_B Q) + \frac{\gamma}{2}\|P\|_F^2 + c e^\top \xi + \sum \alpha_i (1 - \xi_i - y_i w^\top P^\top x_i)\\[4pt] &= \frac{1}{2} Tr(P^\top M^{-1} P) - Tr(P^\top (\lambda S_B Q + Z \alpha w^\top)) + \alpha^\top e + (c e - \alpha - \beta)^\top \xi \end{align*} \] 上式合并了重构项和 \(P\) 的正则项，使用 \(M^{-1} = \lambda S_A + \gamma I\)，以及将分类约束项 \(\sum \alpha_i (1 - \xi_i - y_i w^\top P^\top x_i)\) 写成了矩阵形式：\(\alpha^\top e - \alpha^\top \xi - Tr(P^\top Z \alpha w^\top)\)。

利用 Karush-Kuhn-Tucker（KKT） 条件简化 \(\xi\) 和约束，先针对松弛变量 \(\xi\) 求偏导： \[ \frac{\partial L}{\partial \xi} = c e - \alpha - \beta = 0 \implies \alpha + \beta = c e \] 因为乘子 \(\alpha, \beta \geq 0\)，这直接给出了 \(\alpha\) 的范围约束： \[ 0 \leq \alpha_i \leq c \] 同时，由于 \(\alpha + \beta = c e\)，拉格朗日函数中的 \((c e - \alpha - \beta)^\top \xi\) 项变为 0。

把在上一阶段求得 \(P\) 的最优解化简为： \[ \begin{align*} P &= (\lambda S_A + \gamma I)^{-1} (\lambda S_B Q + Z \alpha w^\top)\\[4pt] &= MB \end{align*} \] 其中 \(B = \lambda S_B Q + Z \alpha w^\top\)。

将 \(P = M B\) 代入拉格朗日函数中关于 \(P\) 的部分： \[ L_P = \frac{1}{2} Tr((M B)^\top M^{-1} (M B)) - Tr((M B)^\top B) \] 由于 \(S_A\) 是对称矩阵，因此 \(M\) 也是对称矩阵，有 \(M^\top = M\)，简化得： \[ L_P = \frac{1}{2} Tr(B^\top M B) - Tr(B^\top M B) = -\frac{1}{2} Tr(B^\top M B) \] 将 \(B = \lambda S_B Q + Z \alpha w^\top\) 代入： \[ \begin{align*} B^\top M B &= (\lambda Q^\top S_B^\top + w \alpha^\top Z^\top) M (\lambda S_B Q + Z \alpha w^\top)\\[4pt] &= \lambda^2 Q^\top S_B^\top M S_B Q + \lambda Q^\top S_B^\top M Z \alpha w^\top + \lambda w \alpha^\top Z^\top M S_B Q + w \alpha^\top Z^\top M Z \alpha w^\top \end{align*} \] 展开后得到四项：

  1. 第一项： \(\lambda^2 Q^\top S_B^\top M S_B Q\)。这不包含 \(\alpha\)，在优化时是常数。
  2. 第二项： \(\lambda Q^\top S_B^\top M Z \alpha w^\top\)。
  3. 第三项： \(\lambda w \alpha^\top Z^\top M S_B Q\)。
  4. 第四项： \(w \alpha^\top Z^\top M Z \alpha w^\top\)。

接下来对上述四项分别求迹，因为第二项和第三项互为转置，因此： \[ Tr(\lambda w \alpha^\top Z^\top M S_B Q) = \lambda w^\top (Q^\top S_B^\top M Z \alpha) = \lambda (Z^\top M S_B Q w)^\top \alpha \] 对于第四项利用 \(Tr(ABC) = Tr(CAB)\)，得到： \[ Tr(w \alpha^\top Z^\top M Z \alpha w^\top) = Tr(w^\top w \cdot \alpha^\top Z^\top M Z \alpha) = \|w\|_2^2 (\alpha^\top Z^\top M Z \alpha) \] 回到一开始的拉格朗日函数： \[ L(P, \xi, \alpha, \beta) = \frac{1}{2} Tr(P^\top M^{-1} P) - Tr(P^\top (\lambda S_B Q + Z \alpha w^\top)) + \alpha^\top e + (c e - \alpha - \beta)^\top \xi \] 刚刚的化简和求迹都是针对 \(L_P\) 进行的，\(\alpha^\top e\) 被放在了一边，现在计算完成后需要将结果和 \(\alpha^\top e\) 合并： \[ \begin{align*} D(\alpha) &= -\frac{1}{2} \left( \|w\|_2^2 \alpha^\top Z^\top M Z \alpha + 2 \lambda (Z^\top M S_B Q w)^\top \alpha + \text{const} \right) + e^\top \alpha\\[4pt] &= -\frac{\|w\|_2^2}{2} \alpha^\top (Z^\top M Z) \alpha + (e - \lambda Z^\top M S_B Q w)^\top \alpha + \text{const} \end{align*} \] 最大化这个对偶函数 \(D(\alpha)\)，即得到了论文中的 QPP 问题： \[ \max_{\alpha} -\frac{\|w\|_2^2}{2} \alpha^\top Z^\top M Z \alpha + (e - \lambda Z^\top M S_B Q w)^\top \alpha, \quad \text{s.t. } 0 \leq \alpha_i \leq c \] 这里的二次项 \(-\frac{\|w\|_2^2}{2} \alpha^\top (Z^\top M Z) \alpha\) 反映了样本在经过 \(M\) 缩放处理后的相关性，\(\|w\|_2^2\) 的存在说明分类器的权重直接影响了子空间的拉伸。一次项 \(e^\top \alpha\) 表示标准 SVM 要求样本离开超平面，而减去的 \(\lambda Z^\top M S_B Q w\) 部分则是来自重构误差的修正。它告诉模型，如果某个样本为了重构而必须待在某个位置，那么 SVM 对它的分类约束可以稍微“妥协”一点。

第三阶段：固定投影矩阵和 SVM，优化重构字典

当 \(P\) 和 \(w\) 固定时，目标函数只剩下与重构相关的部分。

这里把最开始目标函数的简洁形式搬过来： \[ \sum_{i,j=1}^n \|x_i - Q P^\top x_j\|_2^2 G_{ij} = Tr(P^\top S_A P) - 2 Tr(P^\top S_B Q) + \text{const} \] 在这一阶段要优化的重构形式为： \[ \max_{Q} Tr(P^\top S_B Q), \quad \text{s.t. } Q^\top Q = I \] 这是一个经典的正交普鲁克分析（Orthogonal Procrustes Problem），它有闭式解（Closed-form solution）：
